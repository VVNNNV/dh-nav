# 大海导航 (DH Nav)

基于 WordPress 的 AI 工具导航站主题，使用 PHP + Tailwind CSS 构建现代化前端界面。

## 在线演示

- **演示站点**：https://www.dhjc.net

## 视频教程

- [B站视频安装教程](https://www.bilibili.com/video/BV1Hmo4BsEQm/)
- [YouTube 安装教程](https://youtu.be/gGPhzdiNeIk)

## 功能特性

- 自定义文章类型 `ai_tool`（AI 工具）
- 自定义分类法 `ai_category`（层级分类）、`ai_tag`（标签）
- REST API 支持
- 响应式设计，支持移动端
- 热门推荐两行自适应
- 分类导航两行自适应
- 网站收录提交
- 后台管理面板

## 安装

1. 将 `dh-nav` 文件夹上传到 WordPress 的 `wp-content/themes/` 目录
2. 在 WordPress 后台 -> 外观 -> 主题 中激活"大海导航"
3. 激活后会自动创建演示数据

## 自定义字段

| 字段 | 说明 |
|------|------|
| `tool_url` | 工具网址 |
| `tool_icon` | 工具图标（emoji） |
| `tool_hot` | 是否热门（1/0） |
| `tool_order` | 排序权重 |
